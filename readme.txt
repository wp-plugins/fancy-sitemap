Fancy sitemap generates a javascript/svg sitemap.

== Description ==

This plugin allows you to turn any pages or menu into a javascript/svp rendered sitemap.

Screenshots: <a href="http://www.bunchacode.com/programming/fancy-sitemap/" target="_blank">http://www.bunchacode.com/programming/fancy-sitemap/</a>

Tested On: Firefox 5, Chrome 13, IE 7 and IE 8

== Installation ==

1. Upload `fancy-sitemap` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
